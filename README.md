
# FAETERJ-Rio LaTeX - FAETERJ-Rio-ABNTEX2

### Modelo LaTeX para criação de Trabalhos de Conclusão de Curso da Faculdade de Educação Tecnologica do Estado do Rio de Janeiro

O modelo foi desenvolvido inicialmente por Filipe Rodrigues, no intúito de oferecer uma possibilidade de utilização simplificada do LaTeX na FAETERJ-Rio. 

## O Modelo Contém

### 1. Modelo Puro 

- Consiste no mínimo necessário para se iniciar um projeto com o modelo aqui desenvolvido.

### 2. Modelo com Exemplo

- Consiste no mesmo modelo, entretanto foi incluído nele arquivos que podem ser utilizados como guia, sendo uma espécie de trabalho para referência de uso.

## Modo de Uso

### Começando do Zero (recomendado)

- Baixe o modelo no formato `ZIP` ou use o comando `git clone https://github.com/FilipeRodrigues3003/faeterj-rio-latex.git` 
- Crie uma e acesse em: `https://www.overleaf.com/`
- Crie um `novo projeto` no overleaf com o nome de TCC
- Vá até a opção de `Upload` e inclua os arquivos do `Modelo Puro` baixado do GitHub
- Os arquivos incluidos serão:
    - INFO.tex > `Aqui você colocará as informações pessoais referentes ao TCC`
    - main.tex > `Aqui você incluirá o conteúdo do TCC, como capitulos, e referências bibliográficas`
    - faeterj-rio-abntex2 > `Classe com as modificações da abntex2 para utilização da FAETERJ-Rio (Não deve ser alterado)`

### Adaptação/Atualização de um Projeto já Existente

- Baixe o modelo no formato `ZIP` ou use o comando `git clone https://github.com/FilipeRodrigues3003/faeterj-rio-latex.git` 
- Incluia os arquivos:
    - INFO.tex > `Aqui você colocará as informações pessoais referentes ao TCC`
    - main.tex > `Aqui você incluirá o conteúdo do TCC, como capitulos, e referências bibliográficas`
    - faeterj-rio-abntex2 > `Classe com as modificações da abntex2 para utilização da FAETERJ-Rio (Não deve ser alterado)`
